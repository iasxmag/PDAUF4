# PDAUF

Este proyecto se generó utilizando [Angular CLI](https://github.com/angular/angular-cli) version 19.
La aplicación está desplegada en: https://github.com/iasxmag/PDAUF

## Acceso a la aplicación

La aplicación está alojada en la siguiente web: http://localhost:4200/

```bash
ng serve
```

## Code scaffolding

Angular CLI includes powerful code scaffolding tools. To generate a new component, run:

```bash
ng generate component component-name
```

For a complete list of available schematics (such as `components`, `directives`, or `pipes`), run:

```bash
ng generate --help
```

## Build de producción

Para generar el build:

```bash
ng build
```

This will compile your project and store the build artifacts in the `dist/` directory. By default, the production build optimizes your application for performance and speed.

## Running unit tests

To execute unit tests with the [Vitest](https://vitest.dev/) test runner, use the following command:

```bash
ng test
```

## Running end-to-end tests

For end-to-end (e2e) testing, run:

Angular CLI does not come with an end-to-end testing framework by default. You can choose one that suits your needs.
